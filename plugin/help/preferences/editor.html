Editor Preferences

The Editor settings affect how the TrafficScript editor renders text. It is located in Eclipse's preference dialog under Zeus -> Editor.

General Editor Settings

The general settings for Eclipse's editors also apply to the TrafficScript editor. Click the hyperlink at the top of the Editor page to alter these settings.

Syntax Coloring

The syntax coloring settings change the style and color of the TrafficScript editor's text. Select a text element in the list and un-check the 'Use Default' option to change it, the updated style can be previewed at the bottom of the page.

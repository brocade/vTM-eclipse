The Zeus Viewer

Fig.1: The Zeus Viewer

The Zeus Viewer lists the currently open connections to one or more traffic managers. This is presented as a tree structure, listing the rules associated with a Zeus project underneath it. If you are connected to one or more traffic manager, you will be able to view and manipulate the rules associated with them here.

What are the two icons at the top?

Fig.2: Zeus Viewer Icons

There are two icons at the top of this pane each with a green cross symbol, one to add a new Zeus project, and one to add a new rule. More details of these can be found at Adding New Rules and Connecting to a Traffic Manager. You can also perform these operations using the right click menu.

What is the Navigator tab?

This is an alternative view of the same information, presented as a file system. Please see the appropriate section on the Perspective help page for more information.

rule.getName( )

Returns the name of the currently executing rule. 

$rulename = rule.getname();

rule.getState( )

Returns the state of the currently executing rule, either "REQUEST", "RESPONSE" or "GLBRESPONSE". 

$rulestate = rule.getstate();

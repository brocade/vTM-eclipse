The Editor Window

The editor operates much in the same way as any other standard graphical editor, with all the usual highlighting, cut, copy and paste functionality. However, the Zeus plug-in includes additional capabilities such as syntax checking and auto-completion. See this page for more details.

What happens when saving a rule

Saving the rule can be done from the File menu, or by using the <CTRL>-<s> key combination. At this point, the plug-in writes the rule to the traffic manager, and returns the results of a syntax check. Should there be any errors or warnings, these will be highlighted accordingly. The option to revert the changes is made available at the bottom of the editor. NOTE: reverting will result in the loss of any changes made since the rule was last saved without problems.
